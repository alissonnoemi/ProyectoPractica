Que es git

GitHub es una plataforma en línea que permite a los desarrolladores: Crear, distribuir y mantener software Almacenar y compartir código Trabajar en proyectos de forma colaborativa Establecer relaciones profesionales Promover su trabajo GitHub se basa en el software de código abierto Git, que es un sistema de control de versiones que realiza un seguimiento de los cambios en los archivos. Algunas características de GitHub son: Permite crear repositorios de código y guardarlos en la nube Facilita el intercambio de archivos de código Permite trabajar en proyectos colaborativos de código abierto Permite realizar un seguimiento de quién cambió qué Permite a otros usuarios revisar el código y realizar sugerencias para mejorarlo Permite crear un perfil y una marca para promoverse como profesionales
