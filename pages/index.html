Que es Git

GitHub es una plataforma en línea que permite a los desarrolladores crear, almacenar, compartir y colaborar en proyectos de software: Almacenamiento Los desarrolladores pueden crear repositorios de código y guardarlos en la nube. Colaboración Los usuarios pueden trabajar juntos en el código, revisar los cambios y hacer sugerencias para mejorarlo. Seguimiento GitHub ayuda a realizar un seguimiento de quién cambió qué. Red profesional Los desarrolladores pueden establecer relaciones, colaborar y promover su trabajo. GitHub se basa en el software de código abierto Git, que es un sistema de control de versiones que realiza un seguimiento de los cambios en los archivos.
